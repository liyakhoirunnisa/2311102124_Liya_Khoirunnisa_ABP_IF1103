# Web Profil Pribadi - Liya Khoirunnisa

Website profil pribadi yang responsif dan modern, dibangun menggunakan **Vue 3** dan **Vite**. Proyek ini merupakan hasil migrasi dari web statis menjadi aplikasi _Single Page Application_ (SPA) untuk menampilkan profil, keahlian teknis, riwayat pendidikan, serta galeri proyek digital.

## 🚀 Fitur Utama

- **Dynamic Typing Effect**: Animasi teks yang berubah secara otomatis pada bagian Hero menggunakan logika reaktif Vue.
- **Project Gallery with Filter**: Menampilkan daftar karya pilihan dengan sistem "Show More" untuk menjaga kerapian antarmuka.
- **Interactive Contact Form**: Formulasi pengiriman pesan yang dilengkapi dengan status pengiriman sukses secara _real-time_.
- **Skills Proficiency Bar**: Visualisasi tingkat kemahiran teknologi dengan animasi _loading_ saat halaman diakses.
- **Education Timeline**: Ringkasan riwayat akademis yang disusun secara kronologis.
- **Responsive Layout**: Desain yang sepenuhnya adaptif untuk perangkat mobile, tablet, maupun desktop.

## 🛠️ Tech Stack

- **Framework**: Vue.js 3 (Composition API).
- **Build Tool**: Vite.
- **Styling**: Modern CSS3 dengan variabel kustom dan _scoped styles_.
- **Design Tools**: Figma.
- **Icons**: Custom SVG Icons.
